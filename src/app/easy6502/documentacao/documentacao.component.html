Introdução

Esta plataforma utiliza um emulador javascript baseado na arquitetura do processador 6502 utilizado no console Atari2600.
A estrutura básica das instruções é a seguinte: 

label: instrução operandos ;Comentário

De maneira simplificada o emulador Easy6502 possui algumas particularidades, como por exemplo:

  • O endereço de memória $fe contém um novo byte aleatório a cada instrução.
  • O endereço de memória $ff contém o valor ASCII da última tecla pressionada.
  • Os endereços de memória de $200 até $5ff mapeiam os pixels da tela.
  • Diferentes valores salvos nestes endereços desenham diferentes cores na tela.

As cores suportadas são: 

  $0: Preto
  $1: Branco
  $2: Vermelho
  $3: Azul céu
  $4: Roxo
  $5: Verde
  $6: Azul
  $7: Amarelo
  $8: Laranja
  $9: Marrom
  $a: Vermelho claro
  $b: Cinza escuro
  $c: Cinza
  $d: Verde claro
  $e: Azul claro
  $f: Cinza claro

Sintaxe

Os principais modos de endereçamento são: 

  "#" para valor absoluto
  "$" para endereço em hexadecimal
  "$, X" para endereço + X, em hexadecimal
  "#$" para valor absoluto em hexadecimal
  "#%" para valor absoluto em binário

Snake6502

Neste tópico será explicado o código de exemplo do jogo snake. 

  ; Direções (cada uma utilizando um bit diferente)
  define movingUp      1
  define movingRight   2
  define movingDown    4
  define movingLeft    8

  ; Valor ASCII das teclas para controlar a snake
  define ASCII_w      $77
  define ASCII_a      $61
  define ASCII_s      $73
  define ASCII_d      $64

  ; Variáveis do sistema
  define sysRandom    $fe ;Byte aleatório
  define sysLastKey   $ff ;Última tecla pressionada
